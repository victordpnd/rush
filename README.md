# Music Catalog

A lightweight Python application for managing a personal music collection.

## Overview

Music Catalog is a simple console application that stores songs, displays a collection, and generates listening statistics.

The project demonstrates modular Python development, lists, dictionaries, sorting, and formatted console output.

## Features

- Store songs
- Display music catalog
- Count tracks
- Average rating
- Longest track
- Artist statistics
